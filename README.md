# 📚 Library Management System

A full-stack Library Management System built with **Spring Boot**, **React (Vite)**, and **MySQL** to streamline library operations such as book management, member management, issuing and returning books, and tracking library records.

---

## 🚀 Features

- 📖 Book Management
  - Add, update, delete, and search books
  - Manage book availability

- 👥 Member Management
  - Register and manage library members
  - View member details

- 🔄 Book Issue & Return
  - Issue books to members
  - Return books and update availability
  - Track issued books

- 🔐 Authentication & Authorization
  - Secure login
  - Role-based access (Admin/User)

- 📊 Dashboard
  - Overview of books, members, and issued books
  - Real-time statistics

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/your-username/Library-Management-System.git
```

### Backend

```bash
cd source/Backend-springboot
mvn spring-boot:run
```

### Frontend

```bash
cd source/Frontend-Vite
npm install
npm run dev
```
